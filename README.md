# stdio-ipc

Small helpers for stdio-based request/response with Go and Python implementations.

## Repository layout
- `go/`: Go module (`github.com/olimci/stdio-ipc/go`)
- `python/`: Python package (`ipc`)

## Go
Install:

```sh
go get github.com/olimci/stdio-ipc/go
```

Import:

```go
import "github.com/olimci/stdio-ipc/go"
```

## Python
See `python/README.md` for local development and usage.
